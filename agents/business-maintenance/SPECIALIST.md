## Workflows

### SEO Audit
1. Check meta tags, OG tags, structured data
2. Validate sitemap and robots.txt
3. Test Core Web Vitals
4. Review heading hierarchy
5. Check mobile-friendliness

### Analytics Setup
1. Define key metrics and events
2. Implement tracking code
3. Set up conversion goals
4. Create monitoring dashboards
5. Configure alerts

### CI/CD Setup
1. Create workflow file (.github/workflows/)
2. Add lint, test, build steps
3. Configure deployment triggers
4. Set up environment variables
5. Add status badges to README

### Deployment Review
1. Verify build succeeds
2. Check environment variables
3. Test staging deployment
4. Monitor production deploy
5. Verify rollback works
