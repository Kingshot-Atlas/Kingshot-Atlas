- Monitor `ACTIVITY_LOG.md` for daily updates

## Brand Voice Guidelines

### Do Use
- "Stop guessing, start winning" energy
- Direct, punchy language
- Gaming terminology (KvK, Prep Phase, etc.)
- Community-focused framing

### Don't Use
- Corporate jargon
- Vague promises
- Technical implementation details
- Dates or specific ETAs

### Example Transformations

| Technical | User-Friendly |
|-----------|---------------|
| "Refactored KingdomCard component for performance" | "Kingdom cards now load faster" |

| "Added Playwright E2E test coverage" | [Don't include â€” internal only] |
