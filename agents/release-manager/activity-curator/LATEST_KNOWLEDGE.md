## Gotchas

### Things to Watch
- Don't announce features before they're actually shipped
- Verify with FEATURES_IMPLEMENTED.md before claiming something is live
- Check PARKING_LOT.md before teasing features (might be deferred)
- Coordinate with Discord Community Manager on posting timing

### Common Mistakes
- Including too much technical detail
- Promising specific dates
- Hyping minor changes
- Forgetting to attribute community suggestions
