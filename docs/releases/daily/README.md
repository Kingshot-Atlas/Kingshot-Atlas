# Daily Updates

This directory contains daily update files generated by the Activity Curator.

## Schedule

- **Time:** 02:00 UTC daily
- **Channel:** Discord #patch-notes
- **Format:** Embedded message with categorized changes

## File Naming

```
YYYY-MM-DD.md
```

Example: `2026-01-30.md`

## Template

Each daily update should contain:

1. **Highlight** — The most impactful change
2. **What's New** — New features shipped
3. **Improved** — Enhancements to existing features
4. **Fixed** — Bug fixes
5. **Coming Soon** — Teaser for upcoming features

## Source Data

Updates are generated from:
- `/agents/project-instances/kingshot-atlas/ACTIVITY_LOG.md`
- Agent worklogs
- Git commits

## Brand Voice

All updates must follow `/docs/BRAND_GUIDE.md`:
- Direct and punchy
- Gaming terminology
- Community-focused
- No corporate jargon
